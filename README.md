Responsive Navbar

Responsive Navbar built with HTML, CSS, and JavaScript.

This project features a modern responsive navigation bar with:

- Desktop and mobile layouts
- Hamburger menu animation
- Slide sidebar navigation
- Overlay to close sidebar
- Smooth transitions
- Clean and simple UI

Preview

Desktop mode:
- Horizontal navigation menu

Mobile mode:
- Hamburger menu
- Sliding sidebar menu
- Overlay tap anywhere to close sidebar
- Animated hamburger icon to X

Technologies Used

- HTML5
- CSS3
- JavaScript

Features

Responsive Design
The navbar automatically changes layout on smaller screens.

Hamburger Animation
The hamburger icon transforms into an "X" when active.

Slide Menu
Mobile navigation slides smoothly from the right side.

Overlay
Background overlay appears when the menu is opened and will close sidebar if clicked.

Smooth Transition
Animated transitions for better user experience.
